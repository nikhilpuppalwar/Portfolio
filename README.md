# Nikhil Puppalwar - Developer Portfolio

A modern, immersive, and interactive developer portfolio website built with **Next.js 15**, **React 19**, and **Tailwind CSS**. This project showcases my work as a Computer Engineering Student, Android Developer, and AI/ML Enthusiast through a premium user interface featuring 3D elements, smooth animations, and glassmorphism design.

## üöÄ Features

- **Immersive 3D Landing Page**: Features a stunning 3D scene powered by [Spline](https://spline.design/), creating a memorable first impression.
- **Modern UI/UX**: Designed with a "dark mode first" aesthetic, utilizing glassmorphism, soft gradients, and rounded layouts for a tech-forward look.
- **Smooth Animations**: powered by **Framer Motion** for scroll reveals, transitions, and interactive elements.
- **Interactive Project Showcase**: A custom-built project carousel with 3D rotation effects, image galleries, and detailed project insights.
- **Dynamic Skills Section**: visual representation of technical proficiency across Web, Android, and AI/ML domains.
- **Responsive Design**: Fully optimized for all devices, ensuring a seamless experience on mobile, tablet, and desktop.
- **Particle Effects**: Interactive background particles for added visual depth.

## üìÇ Project Sections

The portfolio is structured into several key sections, each designed to highlight specific aspects of my professional profile:

### 1. **Hero Section**
   - **Purpose**: Captivates visitors with a 3D interactive scene and a clear value proposition.
   - **Key Elements**: Spline 3D background, animated "Get Started" call-to-action, and a glossy, modern introduction.

### 2. **About Me**
   - **Purpose**: Provides a personal introduction and professional summary.
   - **Key Elements**: A concise bio highlighting my roles (Computer Engineering Student, Android Dev, AI/ML Enthusiast) and passion for technology.

### 3. **Skills & Expertise**
   - **Purpose**: Visualizes technical competencies across different domains.
   - **Key Elements**:
     - **Categorized Skills**: Grouped into "Web Development", "Android Development", "AI/ML", "Languages", and "Tools".
     - **Interactive Badges**: Each skill is represented by a uniform, high-quality icon/logo (e.g., React, Kotlin, TensorFlow) with hover effects for a polished look.

### 4. **Featured Projects**
   - **Purpose**: showcasing my best work with deep interactivity.
   - **Key Elements**:
     - **3D Carousel**: A rotating slider to browse projects.
     - **Detail View**: displaying project descriptions, tech stack (e.g., "Kotlin + Firebase"), key features, and live links/GitHub repositories.
     - **Phone Mockups**: Realistic device frames to display mobile app screenshots.

### 5. **Experience**
   - **Purpose**: details my professional history and internships.
   - **Key Elements**: Timeline or card-based layout showing roles at **SPACE For Early Childhood Education** and **Infosys Springboard**, highlighting specific contributions and technologies used.

### 6. **Licenses & Certifications**
   - **Purpose**: validates my skills with continuous learning achievements.
   - **Key Elements**: A grid of certificates from platforms like **Infosys Springboard**, **NPTEL**, and **Simplilearn** (e.g., "Deep Learning", "Android Development"). Clicking a card opens the PDF certificate.

### 7. **Why Hire Me?**
   - **Purpose**: Summarizes my unique value proposition.
   - **Key Elements**: Three core pillars: **DSA & Core Engineering**, **Mobile Development Expertise**, and **Practical Machine Learning**, each reinforcing why I am a strong candidate.

### 8. **Education**
   - **Purpose**: Outlines my academic background.
   - **Key Elements**: Details my B.Tech degree at **PCCOE** and previous schooling, including grades/CGPA.

### 9. **Contact**
   - **Purpose**: Encourages visitors to get in touch.
   - **Key Elements**:
     - **Direct Links**: Email, GitHub, LinkedIn, and Phone.
     - **Contact Form**: A fully styled form for visitors to send messages directly.

## üì¶ Installation & Setup

Follow these steps to run the portfolio locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
